# 🎬 TALLY NODE - 빠른 시작 가이드

### 2️⃣ 마스터 노드 설정

#### 📡 AP 접속
- SSID: `TALLY_NODE_V1_AP_XXXX`
- 비밀번호: `12345678`

#### 🌐 네트워크 설정
1. **Settings → 네트워크**
2. 외부 WIFI 선택 및 비밀번호 입력
3. 외부 WIFI 접속 후 할당 받은 아이피로 접속 (Ex. 192.168.0.111)
_(마스터가 할당 받은 IP로 접속하기 위해서는 PC 혹은 스마트폰도 동일한 WIFI 접속 필요)_

#### 🎥 스위처 연결
1. **Settings → Switcher**
2. 타입 선택: **ATEM** 또는 **vMix** (OBS 추가 중)
3. IP 주소 및 포트 입력 → 저장
4. 연결 성공 시 **"OK"** 표시

#### 🔑 라이센스 인증 (선택)
- **Settings → 시스템** → 라이센스 키 입력
- ✅ 미인증 시: 리시버 1개만 사용 가능
- ✅ 인증 시: 무제한 사용

---

### 3️⃣ 리시버 노드 설정

- **동기화 코드**: 마스터와 동일하게 설정 (기본값: 0)
- **카메라 번호 변경**: 전면 버튼 **길게 누르기**

---

## 🎛️ 사용 순서

1. **마스터 전원 ON** → 스위처 자동 연결 확인 (**OK**)
2. **리시버 전원 ON**
3. 필요 시 동기화 코드 및 LED 밝기 조절

---

## 🔘 버튼 기능 (현재 버튼 기능 수정중, 설명과 다릅니다.)

| 노드 | 동작 | 기능 |
|------|------|------|
| 공통 | 짧게 누르기 | 화면 전환 |
| 마스터 | 길게 누르기 | 동기화 코드 변경 |
| 리시버 | 길게 누르기 | 카메라 번호 변경 |

---

## 🚦 LED 색상 표시

- 🔴 **RED**: PGM (온에어)
- 🟢 **GREEN**: PVW (대기)
- ⚫ **OFF**: 미사용

---

## ⚠️ 주의사항

- 동기화 코드 변경은 **근거리에서만 가능**
- 마스터와 리시버의 **펌웨어 버전 일치 필수**
- 펌웨어 업데이트 실패 시에도 기기 손상 없음
